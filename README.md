# AWS Crowd Agent

Serverless agent for intelligent auto-scaling, cost optimization, and resource management across AWS infrastructure.

## Features

- Automated Lambda and EC2 scaling based on traffic patterns
- Cost optimization recommendations
- Resource utilization monitoring and reporting
- Slack and email alert integration
- Multi-region support

## Tech Stack

Python, AWS Lambda, CloudWatch, Boto3, DynamoDB

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/AWS-Crowd-Agent.git
cd AWS-Crowd-Agent
pip install -r requirements.txt
```

### Usage

```bash
python deploy.py
```
