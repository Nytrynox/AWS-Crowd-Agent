# Data & Datasets (Original)

Use public datasets to evaluate without training:
- ShanghaiTech (Part A/B) for density benchmarking
- Mall dataset for crowd counts
- UCSD Anomaly Dataset / UCF-Crime for motion anomaly examples

Scripts in this folder help you:
- Extract frames from videos to feed the local demo (`local/run_local_demo.py`)
- Prepare a small clip for RTSP producer tests

No training is required; these datasets are optional for evaluation and demo clips.
